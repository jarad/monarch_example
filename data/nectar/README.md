The nectar data is stored in this directory. 


Create the following subdirectory to store your data

  1. YYYY     (4-digit year)
  2. MM       (2-digit month)
  3. DD       (2-digit day)
  4. Recorder (name)
  
Save the scanned data sheet as 

    site_transect_round.pdf
  
Create the raw data file as 

    site_transect_round.csv
